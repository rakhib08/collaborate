# collaborate

This repo is being used to verify the Codex workspace and commit flow.

Included in this branch:

- `dev-tracker-standalone.html`: a browser-only internal Dev Tracker prototype with validated setup, split GitHub summary/detail loading, local performance summaries, and a beta Team Space view.
- `hello.py`: the original test file kept as-is.

## Quick start

To preview the prototype locally:

1. Open `dev-tracker-standalone.html` in a browser.
2. Sign in as `Manager` to run the first-time setup flow.
3. Add your GitHub org or username, repositories, and team members.

The app is fully client-side, so the GitHub token and synced dashboard cache stay in the local browser.
